## Installation

Here is how you can run the project locally:

1. Clone this repo
    ```sh
    git clone https://github.com/muxalik/gaslur-nft.git
    ```

1. Go into the project root directory
    ```sh
    cd gaslur-nft
    ```

1. That's it!

1. Visit `localhost:5173` in your favorite browser.
